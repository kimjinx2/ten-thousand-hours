# 10,000 시간의 법칙

유저가 분야와 하루 훈련 시간을 입력하면 1만 시간을 채우기 위해 필요한 **훈련 일수**를 계산해주는 웹 페이지.
버튼 클릭 시 로딩 화면이 나타나고, 결과가 출력될 시에 팝업 모달로 응원 메시지를 보여준다.

## 주요 기능

- 분야 / 하루 훈련 시간 입력
- 1만 시간 달성까지 필요한 일수 계산(올림 처리 한다)
- 공유 버튼 / URL클립보드 복사

## 사용 기술

- HTML 5
- CSS3 (Flex, Animation)
- JS
